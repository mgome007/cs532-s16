    </h3>
    <p class="post-stamp">

    </p>
    <div class="feed-item-body">
      <p>I can’t believe it’s been almost a year since you’ve left. </p>

<p><br></p>

<p>Monty, I know you didn’t know me. I was just one of thousand of fans of yours, and I know my story isn’t special, but. If you’re, somehow, able to be reading this right now…thank you. </p>

<p><br></p>

<p>It’s sad that your death spurred my creative spark once more but…it had to be done, right? I got off my ass and started working at my art. I’ve been working at my acting and my costume making and…I hope you’re proud. Of this girl that you never met but you continue to inspire to this day. </p>

<p><br></p>

<p>I know you’d be so proud if you could see what everyone has accomplished. We, Rooster Teeth and the community, made a movie, dude, and it was so, so incredible to see. </p>

<p><br></p>

<p>I know everyone still misses you, Monty. But your inspiration still burns so fucking bright. You’ve helped me a lot, Monty. And I’m sorry I never got to meet you. </p>

<p><br></p>

<p>I’ll keep moving forward. Because I know I still have a far way to go until I’m the artist I want to be. But you’re gonna be with me every step of the way, I know it. </p>

<p><br></p>

<p>Thank you so much. I miss you. I love you. We love you. </p>

<p><br></p>

        window.addEventListener('message', function respondToMessage(event) {

            if (!new RegExp('^https?://auth.roosterteeth.com$').test(event.origin)) return;

            switch (event.data.type) {
                case 'loaded':
                    var origin = 'https://auth.roosterteeth.com';
                    if (RT.state.has('requestToken')) {
                        event.source.postMessage({type:'check-token', requestToken:RT.state.get('requestToken')}, origin);
                    } else {
                        event.source.postMessage({type:'attempt-login'}, origin);
                    }
                    break;
                case 'token':
                    submitLoginToken(event.data.token);
                    break;
                case 'error':
                    showError(event.data.message);
                    break;
            }

            function submitLoginToken(loginToken) {
                $.post("http://roosterteeth.com/global-login", {loginToken:loginToken})
                    .done(function success(response) {
                        if (response.success) {
                            $('.global-login-message').show();
                        } else {
                            showError('Auto-login failed: %s', response.message);
                        }
                    })
                    .fail(function error(response) {
                        showError('Auto-login failed: ', response);
                    });
            }

            function showError() {
                if (!console) return;

                console.error.apply(console, arguments);
            }

        });

        var loginIframe = document.createElement('iframe');
        loginIframe.src = '//auth.roosterteeth.com/login';
        loginIframe.style.display = 'none';
        document.body.appendChild(loginIframe);
