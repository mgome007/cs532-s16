<h2 class='post-title entry-title'>
<a href='http://www.techjunkieblog.com/2015/10/install-centos-part-4-adding-new-user.html'>Install CentOS Part 4 : Adding A New User With useradd and usermod Command</a>
</h2>
<span class='meta_author'>Posted By: Jason Huynh - 8:58 PM</span>
<div class='post-header-line-1'></div>
<div class='post-body entry-content'>
<span id='988301731468008733'><br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
One of the most common task you have to do as a Linux administrator is to add a new user. &nbsp;Especially developers who always wants root access. &nbsp;To accomplish this task you can use the useradd command in the Terminal session. &nbsp;Follow the steps below to add a new user to CentOS.<br />
<br />
1. &nbsp;Switch into the root user using <b>su - </b>command<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-YWCqWJTFLiw/ViWJ99_JRJI/AAAAAAAACYY/vvaEBXjbeHU/s1600/useradd-1.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://3.bp.blogspot.com/-YWCqWJTFLiw/ViWJ99_JRJI/AAAAAAAACYY/vvaEBXjbeHU/s1600/useradd-1.png" /></a></div>
2. Type the following command <b>useradd -s &nbsp;/bin/bash -d /home/john -m john </b>then press "Enter"<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-5jgcMv28lEM/ViWLZi2o5II/AAAAAAAACYk/xhKUrxIEbIM/s1600/useradd-2.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://4.bp.blogspot.com/-5jgcMv28lEM/ViWLZi2o5II/AAAAAAAACYk/xhKUrxIEbIM/s1600/useradd-2.png" /></a></div>
<br />
useradd is the command to add a new user to CentOS, below is an explaination of what each switch means in the command above<br />
<br />
-s --shell SHELL - the login shell of the new account, which is the/bin/bash shell<br />
-d --home-dir HOME_DIR - home directory of the new account, which is /home/john<br />
<br />
3. &nbsp;Type ls <b>/home</b> to list the content of the /home directory and you will see the <b>john</b> directory has been created for the user john<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://2.bp.blogspot.com/-H-ykn4elb9Q/ViWNOaB8WdI/AAAAAAAACYw/cs033hHANvI/s1600/useradd-3.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://2.bp.blogspot.com/-H-ykn4elb9Q/ViWNOaB8WdI/AAAAAAAACYw/cs033hHANvI/s1600/useradd-3.png" /></a></div>
4. &nbsp;We've created a new user name john, but he has no password, to set a password for john we can use the <b>passwd </b>command. &nbsp;Type the following command to assign john a password <b>passwd john &nbsp;</b>type in john's password when prompted with <b>New Password:</b> &nbsp; and confirm the password when prompted with <b>Retype new password:</b><br />
<b><br /></b>
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-RZG5R_rQ4Qc/ViWPgWuVD1I/AAAAAAAACY8/piJth3LBwNk/s1600/useradd-4.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="183" src="http://3.bp.blogspot.com/-RZG5R_rQ4Qc/ViWPgWuVD1I/AAAAAAAACY8/piJth3LBwNk/s640/useradd-4.png" width="640" /></a></div>
5. &nbsp;Even though john has a login, we still don't know what John's full name is to assign a real name to the login john we can use the <b>usermod -c</b> command to assign the login name john a real name<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-gQdQjau39K4/ViWQv-DWnzI/AAAAAAAACZE/se4UM-QAJto/s1600/useradd-5.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://3.bp.blogspot.com/-gQdQjau39K4/ViWQv-DWnzI/AAAAAAAACZE/se4UM-QAJto/s1600/useradd-5.png" /></a></div>
the command above <b>usermod -c "John Wallace" john</b> assigns the name John Wallace to the user name john<br />
<br />

<br />
1. &nbsp;Make sure you have internet connection<br />
2. &nbsp;Click "Start" on your "CentOS" VM<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-57gAVUm3UP0/ViOqfl53zVI/AAAAAAAACVA/UvRpXRdSyYc/s1600/centos-8.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://1.bp.blogspot.com/-57gAVUm3UP0/ViOqfl53zVI/AAAAAAAACVA/UvRpXRdSyYc/s1600/centos-8.png" /></a></div>

3. &nbsp;After the boot messages you will be presented with login prompt. &nbsp;Type in the login name of the Administrator user that you've created on the last <a href="http://www.techjunkieblog.com/2015/07/install-centos-iso-on-virtualbox-part-2.html" target="_blank">blog</a>.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-fHBYnM9GM5g/ViPPUtszFRI/AAAAAAAACWo/ZZEPhyYxFmM/s1600/tjunkie-login.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="106" src="http://4.bp.blogspot.com/-fHBYnM9GM5g/ViPPUtszFRI/AAAAAAAACWo/ZZEPhyYxFmM/s320/tjunkie-login.png" width="320" /></a></div>
<br />
<br />
6. &nbsp;Now type <b>sudo&nbsp;yum groupinstall "GNOME Desktop"</b> without the quotes<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-8wNp5eJdaXE/ViPRMpNMcqI/AAAAAAAACW8/Eyv915uSDvk/s1600/yum-gnome.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://4.bp.blogspot.com/-8wNp5eJdaXE/ViPRMpNMcqI/AAAAAAAACW8/Eyv915uSDvk/s1600/yum-gnome.png" /></a></div>
<br />
<br />
7. &nbsp;Type <b>y </b>when asked "Is this ok [y/d/N]<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-0_fh_ZetCnw/ViPRrj2hHRI/AAAAAAAACXE/-8VUOw841YE/s1600/gnome-confirm.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://4.bp.blogspot.com/-0_fh_ZetCnw/ViPRrj2hHRI/AAAAAAAACXE/-8VUOw841YE/s1600/gnome-confirm.png" /></a></div>
<br />
<br />
8. After the Gnome desktop has been installed type <b>reboot now -r </b>to<b> </b>reboot CentOS<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://2.bp.blogspot.com/-AZhiS9SFjkw/ViQwfIAT1vI/AAAAAAAACXk/zbs72R4PtLM/s1600/reboot-now.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://2.bp.blogspot.com/-AZhiS9SFjkw/ViQwfIAT1vI/AAAAAAAACXk/zbs72R4PtLM/s1600/reboot-now.png" /></a></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
9. &nbsp;After the CentOS has been rebooted, login using the Administrator user and type <b>sudo systemctl set-default graphical.target </b>this step is optional. &nbsp;You only want to do this if you want to start up CentOS in desktop mode.</div>
<div class="separator" style="clear: both; text-align: left;">
<b><br /></b></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://2.bp.blogspot.com/-1jbkDvs8hmo/ViQzFwXsHCI/AAAAAAAACXw/It3KJ39rMUU/s1600/set-default-gui.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://2.bp.blogspot.com/-1jbkDvs8hmo/ViQzFwXsHCI/AAAAAAAACXw/It3KJ39rMUU/s1600/set-default-gui.png" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<b><br /></b></div>

<a href="http://4.bp.blogspot.com/-Y83_EhZUniU/ViQzv_Tb0UI/AAAAAAAACX4/ANO6VWLfBTc/s1600/start-gui.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://4.bp.blogspot.com/-Y83_EhZUniU/ViQzv_Tb0UI/AAAAAAAACX4/ANO6VWLfBTc/s1600/start-gui.png" /></a></div>

<br />
<div class="separator" style="clear: both; text-align: center;">

<br />

The previous <a href="http://www.techjunkieblog.com/2015/07/angularjs-spa-part-1-setting-up-angular.html" target="_blank">blog</a> we setup the Angular-Seed boilerplate template for our SPA application. &nbsp;In this blog we are going to prepare the Angular-Seed template for our SPA application which is going to be a simple shopping list application. &nbsp;The steps below describes the steps to clean up some of the pages in the angular-seed template for our application.<br />
<br />
1. &nbsp;Delete the content of the "index.html" page in the angular-seed main folder. &nbsp; It is located in the main folder of the angular-seed template "/angular-seed<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">

<br />

 &lt;meta charset="utf-8"&gt;
 &lt;title&gt;Shopping List App&lt;/title&gt;
 &lt;link rel="stylesheet" type="text/css" href="app.css"&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/angularjs/1.4.2/angular.min.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="app.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="controller.js"&gt;&lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</pre>
<br />
The code above provides the plumbing for our SPA application. &nbsp;It's written using the HTML5 template format so that it is HTML5 compliant. &nbsp;The app.css contains all the styles associated with the application, the "angular.js" file contains the angularJS library framework. &nbsp;The "app.js" file contains the JavaScript codes that will be used for this application. &nbsp;The "controller.js" file contains the controller for the application.<br />
<br />
AngularJS uses the MVC framework, Model-View-Controller. &nbsp;Basically they all work together to make the application work. &nbsp;In a simple explanation the model is the data, the view is what the users see, and the controller is the flow of the application. &nbsp;The next step is the define our application in the app.js file.<br />
<br />
3. &nbsp;Open the "app.js" file<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/--eO985n_NvA/Vhxw1WODr-I/AAAAAAAACSw/ndXp_52cGSU/s1600/angular-seed-app-js.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://1.bp.blogspot.com/--eO985n_NvA/Vhxw1WODr-I/AAAAAAAACSw/ndXp_52cGSU/s1600/angular-seed-app-js.png" /></a></div>
<br />
4. &nbsp;The source code for the app.js file should look like the following<br />
<br />
<pre class="prettyprint">'use strict';

angular.module('shoppingList',['shoppingList.controller']);
</pre>
<br />
<br />
The code above defines the application name 'shoppingList', and the modules that the application is dependent on ['shoppingList.controller'] in this case the application is dependent on the shoppingList.controller module. &nbsp;In the next step we will create the controller for the shoppingList application<br />
<br />
5. &nbsp;Create a file call "controller.js" and source code for the file should look like the following<br />
<br />
<br />
<pre class="prettyprint">'use strict';

angular.module('shoppingList.controller',[])
 .controller('shoppingListController',["$scope",function($scope){
  $scope.shoppingListName="My Shopping List";
}]);

</pre>
<br />
The code above registers a new module called shoppingList.controller with no dependencies. &nbsp;The controller itself passes in the angular object $scope. &nbsp;The $scope object in angular act as a conduit between the view and the controller. &nbsp;Meaning anything that is defined in the $scope object is accessible in the view (.html) page. &nbsp;In the example above we defined a variable in the $scope object call "shoppingListName". &nbsp;Since it's defined in the controller we can now access the data from our view (index.html) page. &nbsp;Now we are ready to make some changes to our "index.html" page to make this an angularJS SPA application.<br />
<br />
6. &nbsp;Open up the "index.html" file and make sure the source code looks like the following.<br />
<br />
<pre class="prettyprint">&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;
&lt;head&gt;
 &lt;meta charset="utf-8"&gt;
 &lt;title&gt;Shopping List App&lt;/title&gt;
 &lt;link rel="stylesheet" type="text/css" href="app.css"&gt;
&lt;/head&gt;
&lt;body ng-app="shoppingList" ng-controller="shoppingListController"&gt;
{{shoppingListName}}
&lt;script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/angularjs/1.4.2/angular.min.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="app.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="controller.js"&gt;&lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</pre>
<br />
7. &nbsp;Type int the following URL in your browser http://localhost:8000/app and you will see the following<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-e_OE8FonS3E/VhyFkX-J1nI/AAAAAAAACTA/1jsnST0cxq0/s1600/browser-shopping-list.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://1.bp.blogspot.com/-e_OE8FonS3E/VhyFkX-J1nI/AAAAAAAACTA/1jsnST0cxq0/s1600/browser-shopping-list.png" /></a></div>
<br />
<br />
<b>Note:</b> In order to see the application in your browser you must first run the "npm start" in your angular-seed folder first</span>

On the previous <a href="http://goo.gl/95zFMY" target="_blank">blog</a>&nbsp;we went over how to create a new virtual machine for CentOS in VirtualBox. &nbsp;In this blog we will go over how to install the CentOS ISO file on the new virtual machine that we've just created. <br /></div>

<div class="separator" style="clear: both; text-align: left;">

17. &nbsp;In the "NEWTORK &amp; HOST NAME" screen make sure you have the following settings. &nbsp;This is the steps that most people forget to do. &nbsp;CentOS does not enable networking by default you can do it manually on the command line, but doing it during the install is much easier.<br />

<br /></div>
<div class="separator" style="clear: both; text-align: left;">
22. &nbsp;Type in the user's name in the "Full name" form field, the "User name" field will be auto populated, just accept the default user name. &nbsp;Then check the "Make this user Administrator" check mark. &nbsp;Click "Done" to create a new user.</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-2MH8CNu93YY/ViPDG8w0BiI/AAAAAAAACWQ/7_7qo_Vq9vI/s1600/create-user.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="217" src="http://3.bp.blogspot.com/-2MH8CNu93YY/ViPDG8w0BiI/AAAAAAAACWQ/7_7qo_Vq9vI/s640/create-user.png" width="640" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
23. &nbsp;After you finish creating the user you will see "Finish configuration" button. &nbsp;Click on it.</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-p_WY_beRsmA/VZyjBBtLvjI/AAAAAAAACP8/CdULylsmsKY/s1600/centos-23.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="115" src="http://4.bp.blogspot.com/-p_WY_beRsmA/VZyjBBtLvjI/AAAAAAAACP8/CdULylsmsKY/s640/centos-23.png" width="640" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
24. &nbsp;You will get the "Complete" and "Reboot" button, click on the "Reboot" button to reboot the virtual machine</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-zRUQcJVInqQ/VZyj3rIh8YI/AAAAAAAACQQ/GVnJNykBCIY/s1600/centos-25.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="104" src="http://3.bp.blogspot.com/-zRUQcJVInqQ/VZyj3rIh8YI/AAAAAAAACQQ/GVnJNykBCIY/s640/centos-25.png" width="640" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
25. &nbsp;After a system reboot, a text login prompt is displayed</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-3dK6qOPvQfY/VZykfxAY5bI/AAAAAAAACQY/BWii2O0P7lw/s1600/centos-26.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://1.bp.blogspot.com/-3dK6qOPvQfY/VZykfxAY5bI/AAAAAAAACQY/BWii2O0P7lw/s1600/centos-26.png" /></a></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
26. &nbsp;Login with the user that you created during the installation.</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://2.bp.blogspot.com/--8ksn5GEJX8/VZylg-aQ1yI/AAAAAAAACQk/TnniL8EoN38/s1600/centos-27.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://2.bp.blogspot.com/--8ksn5GEJX8/VZylg-aQ1yI/AAAAAAAACQk/TnniL8EoN38/s1600/centos-27.png" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
You now have CentOS installed in your Oracle VirtualBox. &nbsp;In the next blog we will go over how to install a lightweight desktop GUI to our CentOS.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-LvoT7UT1YIM/VZymy8OTctI/AAAAAAAACQs/urPwL4RM29U/s1600/centos-28.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="251" src="http://3.bp.blogspot.com/-LvoT7UT1YIM/VZymy8OTctI/AAAAAAAACQs/urPwL4RM29U/s640/centos-28.png" width="640" /></a></div>
<br />
27. &nbsp;After logging in, type "sudo yum update" to install the latest updates<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-DYg_BffVDIk/ViPDjHQMXcI/AAAAAAAACWY/UyfOrsApUt8/s1600/yum-update.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="354" src="http://1.bp.blogspot.com/-DYg_BffVDIk/ViPDjHQMXcI/AAAAAAAACWY/UyfOrsApUt8/s640/yum-update.png" width="640" /></a></div>

<h2 class='post-title entry-title'>
<a href='http://www.techjunkieblog.com/2015/07/installing-centos-in-oracle-virtualbox.html'>Installing CentOS in Oracle VirtualBox Part 1: Create a New Virtual Machine For CentOS</a>
</h2>
<span class='meta_author'>Posted By: Jason Huynh - 5:25 AM</span>
<div class='post-header-line-1'></div>
<div class='post-body entry-content'>
<span id='6984790338018932280'><br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />
If you want to know your way around <a href="http://www.redhat.com/en/technologies/linux-platforms/enterprise-linux" target="_blank">Red Hat Enterprise Linux</a> distribution, but the you don't have the financial ability to obtain a license.&nbsp; CentOS is your best bet get to the the Red Hat Enterprise Linux official experience.&nbsp; Because CentOS is a binary capatible version of Red Hat Enterprise Linux.&nbsp; Meaning all the things that matters are the same, only the branding and logos are different.&nbsp; CentOS is open sourced and can be downloaded for free.&nbsp; Although it might be a couple of versions behind Red Hat Enterprise Linux.&nbsp; But, you should be able to perform everything you can with CentOS that you can with Red Hat Enterprise.&nbsp; Below is a step by step instruction on how to install CentOS in Oracle VirtualBox.<br />
<br />
<b>Step-By-Step Instructions:</b><br />
<br />
1.&nbsp; Type <a href="http://centos.org/">http://centos.org</a> in your browser<br />
2.&nbsp; Click on the "Get CentOS Now" button<br />
<br />
<div class="separator" style="clear: both; text-align: left;">
<a href="http://4.bp.blogspot.com/-w7mQPPFPd1s/VY6nTFwR9jI/AAAAAAAACIg/hIeoqbMWJwU/s1600/centos-1.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="396" src="http://4.bp.blogspot.com/-w7mQPPFPd1s/VY6nTFwR9jI/AAAAAAAACIg/hIeoqbMWJwU/s640/centos-1.png" width="640" />&nbsp;</a></div>
<div class="separator" style="clear: both; text-align: left;">
3.&nbsp; Click on "DVD ISO"</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-WYodzwiLyig/VY6n1DOiKSI/AAAAAAAACIo/doN0Lncn42Y/s1600/centos-2.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="228" src="http://1.bp.blogspot.com/-WYodzwiLyig/VY6n1DOiKSI/AAAAAAAACIo/doN0Lncn42Y/s640/centos-2.png" width="640" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
4.&nbsp; Select the mirror site that is closest to you, or whatever fancies you, then save it to a folder that you will remember later on.</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://3.bp.blogspot.com/-XzrXl6uX888/VY6oXiF2bgI/AAAAAAAACIw/MmjFKPt0CK4/s1600/centos-3.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="388" src="http://3.bp.blogspot.com/-XzrXl6uX888/VY6oXiF2bgI/AAAAAAAACIw/MmjFKPt0CK4/s640/centos-3.png" width="640" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
4. &nbsp;Now we are ready to create a new machine in Oracle VirtualBox to prep for the installation of CentOS</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
2. Launch VirtualBox and click on the "New" button</div>
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-xx47iooZqKQ/VNe0MX_BsDI/AAAAAAAABBY/4CEVU6yJC7Q/s1600/1.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="VirtualBox New Virtual Machine" border="0" height="203" src="http://1.bp.blogspot.com/-xx47iooZqKQ/VNe0MX_BsDI/AAAAAAAABBY/4CEVU6yJC7Q/s1600/1.png" title="VirtualBox New Virtual Machine" width="400" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
3. The "Create Virtual Machine" dialog box will be displayed, input the following information</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-AI0udUv9D0w/VZTDPB27b2I/AAAAAAAACK8/rKC7az7OSw8/s1600/centos-4.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://4.bp.blogspot.com/-AI0udUv9D0w/VZTDPB27b2I/AAAAAAAACK8/rKC7az7OSw8/s1600/centos-4.png" /></a></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
4. It's important that you select 64-bit system, from the drop down if you want to install a 64-bit version of Red Hat</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
5. Give your virtual machine 2 GB of memory to start off with, you can increase this value later, if needed. &nbsp;Two GB should be enough for most tasks. &nbsp;You don't want to allocate too much memory to your VM machines because it might slow down your host machine.</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-IbiC_n6IkBw/VZTD30wjVUI/AAAAAAAACLE/7R5WR5hvtDk/s1600/centos-5.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://1.bp.blogspot.com/-IbiC_n6IkBw/VZTD30wjVUI/AAAAAAAACLE/7R5WR5hvtDk/s1600/centos-5.png" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
6. &nbsp;Click "Next"</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
7. &nbsp;Select "Create a virtual hard drive now" option</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://4.bp.blogspot.com/-dsvMgnEXzMQ/VNe21NBmFtI/AAAAAAAABBo/uxOEJ2fWuJM/s1600/5.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="400" src="http://4.bp.blogspot.com/-dsvMgnEXzMQ/VNe21NBmFtI/AAAAAAAABBo/uxOEJ2fWuJM/s1600/5.png" width="348" /></a></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
8. &nbsp;Click "Create"</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
9. On this screen you will see the flexibility of VirtualBox you can actually create Microsoft Virtual Machines with VirtualBox, it's the best of both worlds. &nbsp;Select "VDI (VirtualBox Disk Image)" to create a VirtualBox Disk Image. &nbsp;Then click "Next"</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-mFvYxFNvdY0/VNe3hAwkcaI/AAAAAAAABBw/fieax9N6f_0/s1600/6.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="400" src="http://1.bp.blogspot.com/-mFvYxFNvdY0/VNe3hAwkcaI/AAAAAAAABBw/fieax9N6f_0/s1600/6.png" width="378" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
10. &nbsp;Select "Dynamically allocated" option, this will grow your disk as needed. Then click "Next"</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://1.bp.blogspot.com/-0DULqUOlek8/VNe4KZQGs9I/AAAAAAAABB4/eHcqWeJKGJc/s1600/7.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="400" src="http://1.bp.blogspot.com/-0DULqUOlek8/VNe4KZQGs9I/AAAAAAAABB4/eHcqWeJKGJc/s1600/7.png" width="385" /></a></div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
11. The next screen is the "File location and size" screen. The initial disk space is 8 GB, &nbsp;I gave my VM 64 GB, but you can give the amount of disk space that you are comfortable with. &nbsp;After you've allocated the disk space, click the "Create" button.</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: center;">
<a href="http://2.bp.blogspot.com/-Ks7EFojMqGE/VZTEg-YwxkI/AAAAAAAACLM/Bp_zzgwVFZM/s1600/centos-6.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://2.bp.blogspot.com/-Ks7EFojMqGE/VZTEg-YwxkI/AAAAAAAACLM/Bp_zzgwVFZM/s1600/centos-6.png" /></a></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
<br /></div>
<div class="separator" style="clear: both; text-align: left;">
12. Now you have a new virtual machine ready for a CentOS in VirtualBox</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">
</div>
<div class="separator" style="clear: both; text-align: left;">
<br /></div>
<div class="separator" style="clear: both; text-align: center;">

function showpageCount(json) {
    var thisUrl = home_page_url;
    var htmlMap = new Array();
    var thisNum = 1;
    var postNum = 1;
    var itemCount = 0;
    var fFlag = 0;

    var upPageHtml = '';
    var downPageHtml = '';
    for (var i = 0, post; post = json.feed.entry[i]; i++) {
        var timestamp1 = post.published.$t.substring(0, 19) + post.published.$t.substring(23, 29);
        timestamp = encodeURIComponent(timestamp1);
        var title = post.title.$t;
        if (title != '') {
            if (itemCount == 0 || (itemCount % pageCount == (pageCount - 1))) {
                if (thisUrl.indexOf(timestamp) != -1) {
                    thisNum = postNum

            }
