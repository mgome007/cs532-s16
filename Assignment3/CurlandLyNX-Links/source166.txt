      <h4><b>m0dular</b> from the <b><a href="/docs/command_line">command line <i class="fa fa-download"></i></a></b></h4>
    
  
      <pre class="code code-highlighted"><span class="hljs-comment"># first</span>
keybase signup  <span class="hljs-comment"># if you're new, or</span>
keybase login   <span class="hljs-comment"># if you're not.</span>

<span class="hljs-comment"># then</span>
keybase pgp select  <span class="hljs-comment"># if you already have a public key, or</span>
keybase pgp gen     <span class="hljs-comment"># if this is all new to you</span>
</pre>
    

  

      <pre class="code code-highlighted">keybase id m0dular <span class="hljs-comment"># verify identity proofs</span>
</pre>

      <pre class="code code-highlighted">keybase pgp encrypt m0dular -m <span class="hljs-string">'a secret msg'</span>
keybase pgp encrypt m0dular <span class="hljs-operator">-s</span> -m <span class="hljs-string">'a signed secret msg'</span>
</pre>

      <pre class="code code-highlighted"><span class="hljs-comment"># optionally track m0dular, so you don't</span>
<span class="hljs-comment"># have to keep id'ing them, even when</span>
<span class="hljs-comment"># switching machines</span>
keybase track m0dular

<span class="hljs-comment"># and psst, people you track are pulled into GPG too</span>
gpg --encrypt <span class="hljs-comment"># etc.</span>
</pre>
